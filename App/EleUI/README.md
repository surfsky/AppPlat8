# Element Razor Page UI Components

该目录包含了基于 Element Plus 的 Razor Page 组件，用于快速构建 Web 应用的用户界面。包括但不限于：

- EleTable：表格组件
- EleForm：表单组件
- Contol: 控件
- EleContainer：容器组件
- EleUIJs
  - 包含 EleTable 和 EleForm 组件的 JavaScript 实现文件
  - EleTableAppBuilder：辅助类，用于简化 EleTable 组件的 Vue 应用程序挂载过程
  - EleFormAppBuilder：辅助类，用于简化 EleForm 组件的 Vue 应用程序挂载过程
