# 使用 Pencil AI 来创建设计稿，以及输出对应的 HTML/CSS 代码

1. 安装 Pencil AI 插件

2. 创建提示语

- 用 pencil 扩展 mcp 做一个带左侧导航栏的仪表盘页面，使用 Shadcn/UI 组件库，文件位于 /Design/ 目录下
- 导航栏包含首页、用户管理、角色管理、日志管理等模块
- 首页展示系统的统计数据，如用户数量、角色数量、日志数量等
- 用户管理模块展示所有用户的列表，包括用户名、邮箱、注册时间等
- 角色管理模块展示所有角色的列表，包括角色名称、权限等
- 日志管理模块展示系统的操作日志，包括操作时间、操作人、操作内容等

3. 导出提示语

- 将设计稿转换为 HTML/CSS 代码，确保页面布局和组件样式与设计稿一致，使用 Tailwind CSS 框架
- 确保页面响应式设计，能够在不同屏幕尺寸下正常显示
- 将设计稿生成 Vite + Vue + TypeScript 项目，项目位于 /AppPlat8/ 目录下
